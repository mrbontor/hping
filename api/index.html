• hping
Resource
Description
GET /api/ping.php?s=url
Pings $s 3 times if it is a valid URL. Valid URLs are resolved and are only considered valid if the IP begins with "fc". Returns 0 if offline, 1 if online, 2 if invalid or 3 if $s is not set. Deprecated in favor of latency.php.
GET /api/dns.php?s=url
Resolves $s to an IPv6 address, using only root DNS. Returns IPv6 address.
GET /api/latency.php?s=url
Pings $s 3 times if it is a valid URL. Valid URLs are resolved and are only considered valid if the IP begins with "fc". Returns -1 if offline, -2 if invalid or -3 if $s is not set. If $s is online it will return the latency (in milliseconds).
